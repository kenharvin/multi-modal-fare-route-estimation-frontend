# Image Assets

Place your image assets in this directory:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen image (1242x2436)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)
- `welcome-illustration.png` - Welcome screen illustration

## Image Guidelines

- Use PNG format for transparency support
- Optimize images for mobile (keep file sizes small)
- Use appropriate resolutions for different screen densities
- Consider using vector graphics (SVG) where possible
